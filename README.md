# Incident Management Dashboard

## Overview

This Power BI dashboard was developed to analyze incident management performance and operational KPIs.

## Business Objective

Monitor incident trends, SLA compliance and operational performance to support data-driven decision making.

## Tools Used

- Power BI
- DAX
- Power Query

## Key Metrics

- Total Incidents
- SLA Compliance %
- Incident Severity Distribution
- Resolution Trends
- Operational Performance KPIs

## Dashboard Features

- Executive Summary Dashboard
- Operational Analysis Dashboard
- Interactive Filtering
- KPI Monitoring
- Trend Analysis
